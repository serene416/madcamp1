# MyJapan

여행지 추천, 일정 정리, 사진 기록을 제공하는 일본 여행 관련 안드로이드 앱입니다.

## 📱 주요 기능

### 1️⃣ 여행지 추천 (첫 번째 탭)
- 사용자 취향 기반 질문을 통해 여행지 추천
- 질문 리스트:
  1. 도시 vs 자연 (도쿄 / 오사카 / 후쿠오카 / 삿포로 / 나고야)
  2. 온천 좋아 vs 싫어
  3. 눈 좋아 vs 싫어
  4. 하루 종일 쇼핑 좋아 vs 싫어
  5. 절의 차분한 분위기 좋아 vs 싫어
  6. 럭셔리한 여행 좋아 vs 싫어
  7. 사람들이 많이 간 곳 선호 (**도쿄 > 오사카 > 후쿠오카 > 삿포로 > 나고야**)
- 추천 여행지는 무조건 2개
- 다음 탭으로 이동 안내 포함

---

### 2️⃣ 일정/루트 추천 (두 번째 탭)
- 선택한 기간과 장소 기반 루트 추천
- 나만의 이미지 갤러리 구축
- 줄글 설명 및 Google Maps API 연동으로 맛집 리뷰 표시
- 기간 옵션: 3박4일 / 4박5일 / 5박6일
- 장소 선택: 여행지 5개 중 선택 가능

---

### 3️⃣ 사진 기록 (세 번째 탭)
- 카메라 촬영
- 이미지 앨범 형태로 표시 (2열 그리드)
- 사진 클릭 시 확대 및 삭제 가능
- 앱 재실행 시 사진 저장 기능 구현 필요
- CAMERA 권한 필요

---

## ⚠️ 예외 / 제한 사항
