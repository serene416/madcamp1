Madcamp1

여행지 추천, 일정 정리, 사진 기록을 제공하는 안드로이드 앱입니다.

📱 주요 기능

1️⃣ 여행지 추천
- 질문 기반 선택 (도시/자연/온천/눈 등)
- 추천 여행지 출력

2️⃣ 경로 / 일정
- 추천 여행지 기반 경로 제공
- 장소/일정 선택 가능 (추후 지도 API 연동 예정)

3️⃣ 사진 기록
- 카메라 촬영
- 2열 그리드 형태 사진 표시
- 사진 클릭 시 확대 및 삭제 가능
- CAMERA 권한 필요

⚠️ 예외 / 제한 사항
- 카메라 권한 미허용 시 촬영 불가

🛠 Tech Stack
- Kotlin
- Jetpack Compose
- Material3
- Android Activity Result API
