# MyJapan

일본 여행 관련 앱으로, 사용자 취향 기반 여행지 추천, 일정/루트 안내, 사진 기록 기능을 제공합니다.

---

## 📱 구현할 기능

### 1️⃣ 첫 번째 탭: 여행지 추천
- 사용자 취향에 맞는 여행지 추천
- 질문 리스트를 통해 일본 여행지 추천 (추천 여행지 2개 고정)
- 질문 리스트 :
  1. 도시 vs 자연 (도쿄 / 오사카 / 후쿠오카 / 삿포로 / 나고야)
  2. 온천 좋아 vs 싫어 (후쿠오카)
  3. 눈 좋아 vs 싫어 (삿포로)
  4. 하루 종일 쇼핑 좋아 vs 싫어 (오사카)
  5. 절의 차분한 분위기 좋아 vs 싫어 (나고야)
  6. 럭셔리한 여행 좋아 vs 싫어 (도쿄)
  7. 사람들이 많이 간 곳 선호 (**도쿄 > 오사카 > 후쿠오카 > 삿포로 > 나고야**)
- 다음 탭으로 이동 안내 포함

### 2️⃣ 두 번째 탭: 일정/루트 추천
- 선택한 기간과 장소 기반 루트 추천
- 선택한 일정 그대로 루트 추천
- 줄글 설명 및 Google Maps API 연동으로 맛집 리뷰 표시
- 기간 옵션: 3박4일 / 4박5일 / 5박6일
- 여행지 5개 중 선택 가능

### 3️⃣ 세 번째 탭: 사진 기록
- 카메라 촬영 및 이미지 앨범 형태 표시 (2열 그리드)
- 사진 클릭 시 확대 및 삭제 가능
- 앱 재실행 시에도 사진 저장
- 사용자의 취향에 따라 폴더 구분 기능
- CAMERA 권한 필요

## ⚙️ 기술 스택
- **언어:** Kotlin / Java
- **IDE:** Android Studio
- **UI:** Jetpack Compose 또는 XML Layout
- **데이터 저장:** SharedPreferences / Room DB (사진 저장용)
- **API:** Google Maps API (맛집 및 리뷰 표시)
- **버전 관리:** Git & GitHub
- **이미지 처리:** Glide / Coil
- **기타:** Android Camera API, RecyclerView / GridView
